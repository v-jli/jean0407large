Outdated

  

This tab allows users to view a list of documents older than the specified number of days/weeks/months/years. Documents may be filtered according to their name or document type.

 

When you click the Edit document () icon or the name of a document, you will be redirected to the document's editing interface in the Content -> Edit section of CMS Desk. By clicking the Navigate to document () icon, the document gets opened in preview mode in a new browser window. After clicking the icon in the header of the Actions column, a context menu is displayed, offering options for export of listed data to various types of files.
