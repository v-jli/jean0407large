Kentico CMS 6.0 Context Help

General tab

CMS Desk > On-line marketing > Scoring > Score edit:

General tab

  

General tab

  

Here you can edit the current score. Fill in the fields as required and click Save.

 

You can also manually recalculate the score according to your rules (if defined) by clicking the Recalculate link.

 

Display name

The name of the score used in Kentico CMS user interface.

Code name

The name of the score used in the code.

Description

The description of the score.

Enabled

Indicates whether the score is enabled.

Send notification at score

When the score of a contact reaches a given value, a notification is sent to the defined e-mail address.

Notification e-mail address

Allows to define an e-mail address to which score notifications will be sent. Please note that the sender e-mail address is defined in Site Manager -> Settings -> System in the E-mails -> Send e-mail notifications from setting.

Schedule rebuild

If enabled, the current score is automatically recalculated in accordance with the defined rules. The Schedule rebuild task has the following properties:

•Period - allows to define the scheduled task period (Second, Minute, Hour, Day, Week, Month, Once).
•Start time - allows to define the scheduled task start time.
•Every - allows to define the scheduled task frequency based on the Period property selection.
•Days - allows to define the days of the week when the scheduled task is run.

 

The Score info box gives you information about the current score status and last recalculation time.

 

Localization expressions, i.e. Localize other languages (), Remove localization () and Localize (), are described in detail in the Localization expressions topic in the Development -> UI cultures and localization section of the Developer's Guide.
