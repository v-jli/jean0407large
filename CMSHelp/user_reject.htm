CMS Site Manager > Administration > Users > Waiting for approval:

Reject user(s)

  

Reject user(s)

  

This dialog allows the rejection of a registered user to be confirmed after the Reject () action was used.

 

If you wish to send an e‑mail informing the user about the reason they were rejected from the site, ensure that the Send e-mail box at the bottom is checked and enter the content into the Reason editing area.

 

Complete the rejection using the Reject button.
