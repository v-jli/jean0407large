Language versions comparison

  

This toolbar is used to control language versions comparison mode. You can switch to the mode by selecting a document in the content tree and clicking the Compare () button in the Language toolbar. It results in the language versions comparison toolbar being displayed, while the document editing area below the toolbar gets split into two parts, each of them displaying a different language version of the document. Both displayed language versions can be edited independently at the same time.

 

On the language versions comparison toolbar, you can find the following controls:

 

Horizontal layout

Splits the editing area horizontally. The top part displays the document in the language selected using the Language toolbar, while the bottom part displays the language selected by the drop-down list in the language versions comparison toolbar.

Vertical layout

Splits the editing area vertically. The left part displays the document in the language selected using the Language toolbar, while the right part displays the language selected by the drop-down list in the language versions comparison toolbar.

Synchronize scrollbars

If enabled, scrollbars in both parts of the editing area will be synchronized, resulting in the same sections of both pages being displayed while scrolling.

Comparison language drop-down list

Using this drop-down list, you choose which language version of the currently selected document will be compared with the language version selected in the Language toolbar.

Close comparison mode

Closes the comparison mode and returns to the standard view of a single language version. The same can be achieved by clicking the Compare () button in the Language toolbar again.

 

For more information on editing of multilingual content, please refer to Developer's Guide -> Content management -> Multilingual content.
